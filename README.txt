Dans ce projet nous allons réaliser une voiture inteligente capable de se déplacer dans une piece 

*****************************************************************************

La voiture sera controlée par un µprocesseur Orange Pi zero 2W et communiquera à l'ordinateur par une transmission sans fil sur un rasberry pico.

Le Orange Pi fonctionne sur Orange Pi 1.0.0 Bulleye avec Linux 5.4.125


*****************************************************************************

Dans les dossiers seront disponnibles les programmes et d'autres informations importantes

*****************************************************************************
