# Projet de Voiture Intelligente 

Dans ce projet je vais réaliser une voiture intelligente capable de se 
déplacer seul dans une pièce.

La voiture sera contrôlée par un µprocesseur Orange Pi zero 2W.

## Contenu

Documentations

Pièces 3D

Programmes Arduino

Programme Orange Pi
